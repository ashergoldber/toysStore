Users API:

Method Route Description Request Body
GET "/" Test endpoint to check if the server is running

  • NaN

  • http://localhost:3003
GET "users/" Returns all users without password

  • NaN

  • http://localhost:3003/users
GET "users/userInfo" Get the information for the authenticated user (requires authentication)

Authenticate JWT token (in request header)
POST "users/signUp" sign up / create user

{name: String, email: String, password: String} *Required all

POST "users/login" logIn to get authenticate token

{email: String, password: String} *Required all

Toys API:

Method Route Description Parameters / Request Body
GET "/toys" Returns a list of toys according to the query parameters Default: sends the last 6 toys (6 per page)
http://localhost:3003/toys
  • page (optional): return page number (default: 1). http://localhost:3003/toys?page=1
  • perPage (optional): number of toys per page (default: 6). http://localhost:3003/toys?perPage=6
  • sort (optional): return toys by sort parameter (default: id). http://localhost:3003/toys?sort=_id
  • reverse (optional): if set to yes, sorts in descending order. http://localhost:3003/toys?category=
  • s (optional): return toys by name or info name. http://localhost:3003/toys?s=
  • cat (optional): return toys by category name. http://localhost:3003/toys?cat=
GET "toys/price" Returns a list of toys filtered by price range.
  • min (optional): minimum price to filter by (default: 0).
  • max (optional): maximum price to filter by (default: infinity).
    • http://localhost:3003/toys/?min=0&max=99999
GET "toys/single/:id" Returns a single toy by its ID
  • id the ID of the toy to retrieve. http://localhost:3003/toys/single/6465fe7d807dc0bc99deaa26
POST "toys/" Create / Add a new toy *Token Required
  • Token (in header)
  • {name: String, info: String, category: String, img_url: String, price: Number} (img_url not required)
DELETE "toys/:idDel" delete a toy by its ID *Token Required
  • idDel personal ID toy to delete
PUT "toys/:idEdit" update a toy by its ID *Token Required
  • idEdit personal ID toy to delete
